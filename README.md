# python-GUI-basic-calculator
A high-resolution (1000x1000) Tkinter calculator

## Features
- **Large Canvas:** Optimized for high-resolution displays.
- **Power Function:** Includes exponentiation (`^`).
- **Dark Mode:** High-contrast charcoal and white UI.

## How to Run
1. Install Python 3.
2. Run `python BasicCalculator.py`.
